# Archived plans

Historical implementation plans for work that has shipped (`status: completed`)
or been superseded by a later plan (`status: superseded`). Kept for context and
traceability; they record decisions as they were made and may use older naming
(e.g. the "Upwell" product name from before the rename to Risezome).

Active plans, if any, live one level up in `docs/plans/`.
